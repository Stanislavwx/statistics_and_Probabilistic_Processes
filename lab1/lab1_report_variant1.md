# Лабораторна робота 1

## Тема

Моделювання процесу відносно-частотного визначення ймовірності.

## Мета роботи

За допомогою моделі «дослід - подія» ознайомитися з відносно-частотним визначенням ймовірності та перевірити, як частота появи події наближається до теоретичної ймовірності при збільшенні числа дослідів.

## Що вимагає методичка

Після аналізу PDF для лабораторної роботи 1 потрібно:
1. Згенерувати вибірки цілих псевдовипадкових чисел.
2. Для кожного значення N побудувати гістограму частот випадання k-го числа.
3. Повторити моделювання для N = 10, 100, 1000, 10000.
4. Проаналізувати результати та зробити висновки.

## Вихідні дані

- Варіант: 1.
- Діапазон випадкових чисел: від 1 до 4.
- Модель: підкидання правильної фігури з 4 гранями.
- Теоретична ймовірність випадання кожної грані: P = 1/4 = 0.2500.
- Обсяги вибірок: 10, 100, 1000, 10000.
- Використаний генератор: `random.randint()` у Python 3.
- Для відтворюваності використано зерно: 42.

## Аналіз результатів

При малому числі дослідів частоти помітно відрізняються від теоретичної ймовірності. Це нормально, тому що випадкові коливання ще великі.

Збіжність частот до теоретичної ймовірності не обов'язково є строго монотонною на кожному кроці. Наприклад, для окремої випадкової вибірки відхилення при N = 100 може бути не меншим, ніж при N = 10.

Але для великих вибірок, особливо при N = 1000 і N = 10000, стовпці гістограм вирівнюються, а частоти стають ближчими до теоретичного значення. Це підтверджує відносно-частотне означення ймовірності.

## Висновки

У роботі змодельовано підкидання правильної фігури з 4 гранями для чотирьох обсягів вибірки. У міру збільшення кількості дослідів частоти загалом наближалися до значення 0.2500.

Отже, експеримент підтверджує, що при збільшенні числа випробувань відносна частота появи події прямує до її ймовірності.

## Відповіді на контрольні питання

1. Дослід - це дія, яка завершується певним результатом. Подія - це результат випадкового досліду, який стає відомим після його завершення.
2. Достовірна подія обов'язково відбувається. Випадкова подія може відбутися або не відбутися. Сумісні події можуть відбутися одночасно, а несумісні - ні.
3. Частота появи події A в серії випробувань - це відношення числа появ події N_A до загальної кількості дослідів N: v_A = N_A / N.
4. Ймовірність події A - це границя її відносної частоти при необмеженому збільшенні числа дослідів.
5. Для несумісних подій імовірність суми дорівнює сумі імовірностей: P(A1 + A2) = P(A1) + P(A2).
6. Якщо події утворюють повну групу несумісних подій, то сума їх імовірностей дорівнює 1.
7. Умовна ймовірність - це ймовірність події за умови, що інша подія вже відбулася. Апріорна ймовірність задається до досліду, а апостеріорна визначається після врахування додаткової інформації.
